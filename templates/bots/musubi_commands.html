Phone

The core of Musubi — place calls, end them, and control how you appear.

Core

Search for another server to connect with. Musubi dials into the network and finds a partner — premium guilds get priority matching. Calls auto-end after 30 minutes of silence.

USAGE /call
Core

End the current call or cancel an in-progress search before a match is found.

USAGE /hangup
Core

Toggle anonymous mode on or off. When enabled, you appear as 📞 Anonymous on the other server instead of your real name and avatar.

USAGE /anonymous
Core

Send your Discord tag to the other server during an active call so they can add you after the call ends.

USAGE /friendme
👤

Profile

Control how you appear to other servers during calls.

Profile

View your current profile — premium status, custom nickname, avatar, and whether anonymous mode is on.

USAGE /me status
✨ User Premium

Set a custom display name that other servers see during calls instead of your Discord username.

USAGE /me name [name]
EXAMPLE /me name spector
✨ User Premium

Set a custom avatar URL that other servers see during calls instead of your Discord profile picture.

USAGE /me avatar [url]
EXAMPLE /me avatar https://i.imgur.com/abc.png
Profile

Wipe your custom display name and avatar. Reverts back to your Discord defaults on all future calls.

USAGE /me reset
⚙️

Server

Admin commands for registering and configuring your server on the network.

Admin

Register your server on the Musubi network and pick a booth channel — the single channel where all calls happen.

USAGE /setup
Admin

Change the booth channel after initial setup without needing to re-register your server.

USAGE /setbooth [#channel]
EXAMPLE /setbooth #phonebooth
Admin

Set a custom command prefix for your server. Applies to all members in this server.

USAGE /prefix server [prefix]
EXAMPLE /prefix server !
Destructive

Remove your server from the Musubi network entirely. Two-step confirmed — no accidents.

USAGE /unregister
✨

Premium

User Premium unlocks custom identities and personal prefix. Server Premium unlocks attachments, stickers, and double XP.

✨ User Premium

Set a personal prefix that follows you across every server you use Musubi in — not just the current one.

USAGE /prefix self [prefix]
EXAMPLE /prefix self $
Premium

Check what premium is currently active for you personally and for your server.

USAGE /premium status
Premium

Activate a premium key for User or Server Premium. Keys are distributed by KpnWorld.

USAGE /redeem [key]
EXAMPLE /redeem XXXX-XXXX-XXXX
🏆

Callboard

Servers earn XP every message sent on a call. Top 7 reset monthly.

Network

View the current monthly leaderboard. Servers earn XP per message sent during calls — premium servers earn double. Top 7 are displayed and reset each month.

USAGE /callboard
